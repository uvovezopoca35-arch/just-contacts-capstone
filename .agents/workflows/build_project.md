# Build Project

Follow these steps to install dependencies, analyze, and build the Next.js application.

1. Ensure you are in the project directory where `package.json` is located.
2. Install the necessary dependencies.
// turbo
3. Run `npm install`
4. Run Typecheck to catch TypeScript errors.
// turbo
5. Run `npm run typecheck`
6. Compile the codebase for production.
// turbo
7. Run `npm run build`
